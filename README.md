# Smart Study Planner with Dashboard

## Project Overview

Smart Study Planner is a lightweight full-stack web application designed to help students organize subjects, manage study tasks, track deadlines, and monitor progress through an interactive dashboard.

This application demonstrates frontend, backend, and database integration using modern web development technologies.

---

## Features

User Authentication

* Signup
* Login
* Logout

Subjects Module

* Add subjects
* Delete subjects
* View subject list

Tasks Module

* Add study tasks
* Set deadlines
* Mark tasks as completed
* Delete tasks

Dashboard Analytics

* Total Subjects counter
* Total Tasks counter
* Completed Tasks counter
* Pending Tasks counter
* Progress tracking interface
